Quick notes pick rational file name automatically.

File content is stored in Markdown

Wiki-like Camel case linking between documents is supported.

File content is displayed in CSS.

You can make a quick note from a template.

The current date can be included in a template file name automatically - to support Today.txt.

You can search for existing notes using full text search.

You can sort existing notes into folders very quickly.

You can implement a trivial todo list with lines and checkboxes.

You can email the underlying Markdown file.

(Stretch) You can email the rendered CSS/HTMl.
